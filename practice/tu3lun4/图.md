## 图的存储

- 邻接矩阵
- 直接存边
- 邻接表
- 链式前向星
  - head[u]存储以u为起点的头指针
  - edge[i]存储第i条边的信息
  - edge[i].next存储第i条边的下一个边的下标

遍历(dfs)：
```c++
// 邻接表
void dfs(int u){
    if(visited[u]){
        return;
    }
    visited[u] = true;
    for(auto v:adj[u]){
        dfs(v);
    }
}
```
```c++
// 链式前向星
void dfs(int u){
    if(visited[u]){
        return;
    }
    visited[u] = true;
    for(int i=head[u];i!=-1;i=edge[i].next){
        dfs(edge[i].to);
    }
}
```
